В модуле json_to_array.py реализован алгоритм преобразования данных 3D  модели, записанных в формате json  в виде словаря  
в трехмерный массив вида Nx3x3. Здесь N --  число треугольников, а каждый элемент массива представляет собой одну из трех 
координат всех трех вершин треугольной грани. Словарь имеет два ключа vertics, faces, в которых хранится геометрия модели.
